# Security Policy

## Reporting a Vulnerability
If you discover a security vulnerability, please do **not** open a public issue.

Preferred reporting methods:
1. GitHub Security Advisories (if this repository is hosted on GitHub).
2. Email: ma10016@nyu.edu

We will acknowledge receipt within 72 hours and work with you on a fix.

## Supported Versions
We generally support the latest `main` branch. Security fixes will be published as soon as possible.
